
📧 SMS Spam Detection System
🚀 Live Demo: [https://ai-sms-spam-detection.streamlit.app/]
This project is a Machine Learning application designed to classify SMS messages as either Spam or Ham (Legitimate).
✨ Key Features:
User Interface: Built with Streamlit for a clean and interactive web experience.
Algorithm: Uses the Multinomial Naive Bayes algorithm, which is highly effective for text classification.
Vectorization: Implements TF-IDF Vectorizer to convert text data into a numerical format for the model.
📁 Project Files:
app.py: The main application script that runs the web interface.
model.pkl: The pre-trained machine learning model file.
vectorizer.pkl: The saved TF-IDF vectorizer used for text processing.
requirements.txt: A list of necessary Python libraries (Streamlit, Scikit-learn, etc.).
